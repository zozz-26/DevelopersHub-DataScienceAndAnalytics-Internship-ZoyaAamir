# Phase 2: Advanced Data Science Tasks

Advanced internship tasks covering classification, explainable AI (XAI), and unsupervised learning.

---

## Task 1: Term Deposit Subscription Prediction
- Predicted term deposit subscriptions using UCI Bank Marketing Dataset (41,188 records)
- Trained Logistic Regression and Random Forest models
- Random Forest performed best: 91.7% accuracy, 0.59 F1-Score, 0.94 AUC
- Used SHAP (Explainable AI) to explain model predictions — call duration identified as the strongest predictor

## Task 2: Customer Segmentation Using Unsupervised Learning
- Segmented 200 mall customers using K-Means Clustering
- Found optimal clusters (K=5) using Elbow Method
- Visualized multi-feature clusters using PCA
- Identified 5 segments (VIPs, Careful Rich, Impulsive Buyers, Regulars, Budget Customers) with tailored marketing strategies
