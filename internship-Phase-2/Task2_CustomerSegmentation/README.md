## Task 2: Customer Segmentation Using Unsupervised Learning

Clustered mall customers into segments based on income and spending habits using K-Means, and proposed marketing strategies for each segment.

### Approach
- Performed EDA on Mall Customers Dataset (200 records)
- Applied Elbow Method to find optimal number of clusters (K=5)
- Used K-Means Clustering to segment customers based on income and spending score
- Applied PCA to visualize clusters using 3 features (Age, Income, Spending Score)
- Proposed targeted marketing strategies for each segment

### Results & Insights
- Identified 5 distinct customer segments: VIPs (high income, high spending), Careful Rich, Impulsive Buyers, Regulars, and Budget Customers
- VIP segment is the prime target for premium offers and loyalty programs
- High-income low-spenders represent untapped revenue potential
- PCA effectively visualized multi-feature clusters in 2D
